# 🚀 Vercel Deploy Rápido

## 3 Pasos para Desplegar en Vercel

### 1️⃣ Ir a Vercel
```
https://vercel.com
```

### 2️⃣ Importar tu Repositorio GitHub
- Haz clic en **"Add New..." → "Project"**
- Selecciona **"Import Git Repository"**
- Busca: `agrotech`
- Selecciona el repositorio

### 3️⃣ Configurar y Deploy
- Vercel detectará automáticamente `vercel.json`
- Haz clic en **"Deploy"**
- ¡Espera 2-3 minutos!

## 🔄 Actualizaciones Automáticas

Cada vez que hagas `git push` a GitHub:
```bash
git add .
git commit -m "Tu cambio"
git push
```

**Vercel automáticamente redeploya** ✨

---

## 💡 Tip: Ver Dominios Personalizados

Si quieres un dominio propio (ej: www.agrotech.com):

En Vercel → Settings → Domains → Add Domain
